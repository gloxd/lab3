# Задание 1

### Текст задачи

Получить список из максимальных цифр натуральных чисел, содержащихся в
исходном списке (Seq.map)

### Алгоритм для задания 1
1. Получить список чисел от пользователя
2. Преобразовать список в последовательность (Seq)
3. Для каждого числа:
   - Преобразовать число в строку
   - Преобразовать строку в последовательность символов
   - Каждый символ преобразовать в цифру (int)
   - Найти максимальную цифру через Seq.max
4. Собрать результаты в список

# Задание 2

### Текст задачи

Найти суммарную длину строк в исходном списке через Seq.fold.

### Алгоритм для задания 1
1. Получить список строк от пользователя
2. Преобразовать список в последовательность
3. Применить Seq.fold:
   - Начальное значение аккумулятора = 0
   - Для каждой строки: прибавить её длину к аккумулятору
4. Вернуть итоговую сумму

# Задание 3

### Текст задачи

Вывести первый по алфавиту файл в указанном каталоге.

### Алгоритм для задания 1
1. Получить путь к каталогу от пользователя
2. Проверить существование каталога
3. Получить все файлы в каталоге через Directory.GetFiles
4. Преобразовать полные пути в имена файлов
5. Отфильтровать null-значения
6. Отсортировать по алфавиту через Seq.sort
7. Взять первый элемент через Seq.tryHead
8. Вывести результат или сообщение об ошибке
